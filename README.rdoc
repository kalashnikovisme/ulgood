Ulgood.ru

Дизайн сайта выполнен по техническому заданию.

Для авторизации в социальных сетях используются приложения соответствующих социальных сетей.
Ключи и ID приложений хранятся в /config/initializers/devise.rb. 
Страницы приложений доступны для настройки владельцу приложения (разработчику сайта).
